CSV Upload Format

  
Please upload a CSV file with the following header (column names):


  
  a_comp_1,a_comp_2,a_comp_3,
  a_frac_1,a_frac_2,a_frac_3,
  c_comp_1,c_comp_2,c_comp_3,
  c_frac_1,c_frac_2,c_frac_3,
  f_comp_1,f_comp_2,f_comp_3,f_comp_4,f_comp_5,
  f_frac_1,f_frac_2,f_frac_3,f_frac_4,f_frac_5,
  n_300,T,p_n,ZT
  


  
Explanation of each field:

  
    
    
  • a_comp_*, c_comp_*, f_comp_*: Element symbols (e.g. Sb, Co, Ba) for anion, cation, and filler components.
    • 
    
  • a_frac_*, c_frac_*, f_frac_*: Corresponding atomic fractions (e.g. 1.0, 0.5).
    • 
    
  • n_300: Carrier concentration at 300 K (in cm⁻³).
    • 
    
  • T: Temperature (in K).
    • 
    
  • p_n: Type of carrier (0 = p-type, 1 = n-type).
    • 
    
  • ZT: Dummy value — this field is not used during prediction, but is required for input formatting.
    • 
  


  
Empty fields in the single-system form are automatically treated as 0.

  
You can enter any number for ZT (e.g., 0.0) — it is ignored by the model.
